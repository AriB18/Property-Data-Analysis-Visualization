#  Hassall Grove Property Data Analysis

> **Data Visualization Foundations: Property Investment Analysis for Hassall Grove, NSW (2011-2021)**


---

##  About The Project

This project provides a comprehensive property investment analysis for **Hassall Grove**, a suburb in New South Wales, Australia (Area Code: SAL11873). Using data from the Australian Bureau of Statistics (ABS) and PriceFinder, this analysis examines key demographic, economic, and housing indicators across a 10-year period to help potential investors make informed decisions.

##  Key Findings

###  Concerning Trends

1. **Housing Affordability Crisis**
   - House prices increased by 111.4% while income grew only ~35%
   - Affordability index worsened from 4.48 to 7.03 (household)

2. **Shift to Rental Market**
   - Rental properties increased from 21.1% to 29.0%
   - Mortgage ownership dropped from 61.5% to 49.1%

3. **Rent Stress Tripled**
   - Households paying >30% income on rent: 9.1% → 35.2%

4. **Workforce Casualization**
   - Full-time employment dropped 13.1%
   - Part-time work increased 6.5%

###  Positive Indicators

1. **Unemployment Improved** - Down 1.4% over the decade
2. **Savings Growth** - Median savings up 66.9%
3. **Supply Meeting Demand** - Ratio stable around 1.01
